# Nueva Tarea: hacer que el proyecto se transforme en PWA 


## Descripción:
El proyecto actualmente es una aplicación web, pero se quiere transformar en una Progressive Web App (PWA) para que los usuarios puedan instalarla en sus dispositivos y usarla como una aplicación nativa. para ello se deben realizar los siguientes pasos:

1. Crear un archivo `manifest.json` en la raíz del proyecto con la configuración de la PWA, incluyendo el nombre de la aplicación, el icono, el tema de colores, etc.
2. Crear un archivo `service-worker.js` en la raíz del proyecto con la lógica para cachear los recursos de la aplicación y permitir su funcionamiento offline.
3. Modificar el archivo `index.html` para incluir el enlace al `manifest.json` y registrar el `service-worker.js`.
4. Probar la aplicación en diferentes dispositivos y navegadores para asegurarse de que funciona correctamente como una PWA.
