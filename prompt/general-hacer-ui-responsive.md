# Tarea: hacer app responsive

## Descripción:
Actualmente la aplicación no es responsive, por lo tanto se debe adaptar para que se vea bien en diferentes tamaños de pantalla, como por ejemplo en dispositivos móviles. para ello se deben realizar los siguientes cambios:

1. Utilizar unidades relativas como `%`, `em`, `rem` en lugar de unidades fijas como `px` para los tamaños de los elementos.
2. Implementar media queries en el CSS para ajustar el diseño según el tamaño de la pantalla.
3. Asegurarse de que las imágenes y otros elementos multimedia se escalen correctamente en diferentes dispositivos.
4. Probar la aplicación en varios dispositivos y resoluciones para garantizar una experiencia de usuario consistente.
